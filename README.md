# healthcare（健康モニターデバイス）

## これは何？
体の異常を検知して、健康維持をサポートするデバイス。
心臓の動き・鉄分・水分の3つを同時にチェックする。

## 動作内容
- **スイッチ**：心臓付近に取り付けて、心臓が動いているかを検知する
- **磁気センサ**：体内の鉄分が不足していないかを検知する
- **土壌水分センサ**：体に水分が足りているかを検知する

### 判定ロジック
- もし、スイッチが反応しない（＝心臓が動いていない）状態になったら、警告音を鳴らす
- もし、磁気センサの値が下がった（＝鉄分不足）状態になったら、LEDを赤く光らせる
- もし、土壌水分センサの値が下がった（＝水分不足）状態になったら、LEDを青く光らせて水分補給を知らせる

## 使い方
1. スクーミーのアプリでこのプロジェクトを開く
2. 各センサーを体の指定の位置に装着する
   - スイッチ：心臓付近
   - 磁気センサ・土壌水分センサ：腕など
3. プログラムを実行すると、リアルタイムで体の状態をモニタリングする
