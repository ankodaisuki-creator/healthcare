## じんしんの角の外し方

1. まずじんしんを捕まえます
    * この時じんしんを探す旅に出ます
2. じんしんを見つけたらコントローラーの **上上下下右左右左** を押します
    * バックアップは先に取っておきましょう
    * ~~いかなる可能性があっても当社は保証しません~~
3. じんしんの角が取れます

## じんしんの手懐け方

1. じんしんを探します
    * 食事中に探すと攻撃力が高まるのでやめましょう
2. コントローラーの **LRボタンとAボタンを同時に** 押します。
3. じんしんが喜びます
    * ~~ちょろいとか思っちゃダメだよ~~
